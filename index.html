Repositories

書いたコードの一部は GitHub にて公開しています。アーカイブ化されていないすべてのリポジトリは、わたしがそこから離れていくまで、ずっと保守していきます。技術的な詳細については以下のリンクから見ることができます

PlusArchive

インターネット上にアップロードされているプロモーション用の音楽を集めたサイト。世界中の音楽に耳を澄まし、それを記録していく

tapes

月一でよく聴いている音楽を、特定のアーティストやジャンルに絞って数曲ずつ公開しているサイト。その月に聴いた音楽、その年に聴いた音楽を記録する

blog

技術的な記事をまとめたブログサイト。わたし個人のアウトプットの場であり、ドキュメントでもある。記事は書き捨てではなく、コードと同じようにできる限り更新していく

denene

「控えめな Web」をテーマに、HTML と CSS、そしてほんの少しの JavaScript で何ができるかを確認する個人的な Web サイト。Web におけるセマンティクスやアクセシビリティの実験の場でもある

Other Repositories

ユーティリティ系のツールやデモアプリケーションなどが含まれています。技術的な詳細については以下の外部リンクから見ることができます

ripple

特定の URL からアーティースト名やタイトル、ID や画像などの情報を取得するユーティリティツール。YouTube や SoundCloud、Vimeo、Bandcamp などをサポートしている

Maytalz

Nuxt と Contentful を使った Jamstack なブログのデモサイト。Headless CMS や最新バージョンの Nuxt の動作を確認するときにも使用している

suerte

Next.js と Bootstrap 構成での Storybook の作成と公開をしているリポジトリ。Storybook とその周辺ツールの動作を確認するときにも使用している

Papers

Laravel を使って認証関連の複雑な実装を確認するために作ったリポジトリ。React などを使わない場合の Blade テンプレートやアセット周辺の動作確認にも使用している
